## 翻译指南

### 术语表
| 英文 | 中文 | 备注 |
|------|------|------|
| credits | 积分 | 货币单位 |
| ship | 飞船 | |
| outfit | 装备 | |
| planet | 行星 | |
| system | 星系 | |
| mission | 任务 | |
| cargo | 货物 | |
| hyperdrive | 跃迁引擎 | |
| jump drive | 跳跃引擎 | |
| Republic | 共和国 | 人类政府 |
| Syndicate | 辛迪加 | 企业势力 |
| Free Worlds | 自由世界 | 分裂势力 |
| Hai | 海伊 | 外星种族 |
| Korath | 科拉斯 | 外星种族 |
| Quarg | 夸格 | 外星种族 |

### 翻译原则
1. **保持科幻感** - 使用符合太空歌剧风格的语言
2. **一致性** - 同一术语全文统一
3. **保留占位符** - `<planet>`, `<payment>` 等必须原样保留
4. **自然流畅** - 中文表达要地道，不要直译
