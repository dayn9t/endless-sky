# CLAUDE.md - Claude Code 项目规则

## 重要限制

### 自动提交禁止

**禁止自动提交代码到版本控制系统。**

每次提交之前必须：
1. 明确获得用户确认
2. 用户审查待提交的变更
3. 用户确认提交信息内容
4. 用户明确授权后才可执行 `git commit` 和 `git push`

### 代码修改权限

**未经授权，不得修改任何源程序代码。**

允许修改的范围仅限于：
- ✅ 单元测试代码 (`test/` 目录下的测试文件)
- ✅ 集成测试代码
- ✅ 测试框架和测试工具
- ✅ 测试数据和Mock对象

禁止修改的范围：
- ❌ 核心库代码 (`lib/`)
- ❌ 客户端代码 (`client/`)
- ❌ 服务器代码 (`server/`)
- ❌ 脚本系统 (`scripting/`)
- ❌ AI代码 (`AI/`)
- ❌ 任何生产环境源代码

### 例外情况

如需修改非测试代码，必须：
1. 明确获得用户授权
2. 在提交信息中注明授权上下文
3. 遵循项目的代码审查流程
