# Gamerules
tip "Minimum value"
	`装备或飞船相对于其全新价格所能折旧到的最低价值。所有掠夺、缴获或获赠物品均以此价值为初始值；而所有购入物品则以100%价值起始，并依据其他折旧游戏规则逐步向该最低值折旧。`

tip "Grace period"
	`新购入的装备或飞船开始折旧前必须经过的天数。这使你有短暂时间试用物品，若发现并不喜欢，仍可按全额价格将其售回。`

tip "Maximum age"
	`在宽限期结束后，装备或飞船完全折旧所需经过的天数。`

tip "Daily depreciation"
	`物品每日保值率，即其可折旧价值在每日间保留的百分比。该参数仅影响折旧计算中的指数项。即使设为100%，物品仍会以线性方式持续贬值。`

tip "Spawn attempt period"
	`游戏平均每隔多少帧尝试生成一艘载人飞船。游戏每帧都会在 0 至该值减 1 的范围内随机掷出一个数；若结果为 0，则尝试生成一艘载人飞船。每秒共 60 帧。`

tip "No spawn weight"
	`当尝试生成人物飞船时，此数值代表该次生成失败的“权重”。每种人物飞船都拥有一个与其关联的“权重”，决定其实际生成概率。任一特定人物飞船的生成概率，等于其权重除以所有人物飞船权重之和再加上该数值。这意味着，生成尝试失败的概率即为此数值除以相同的总和。`

tip "NPC max mining time"
	`拥有“采矿”性格的随机生成NPC，将在当前星系内持续采矿指定帧数后，才决定转而执行其他行为。设置该限制的目的，是让玩家能直观看到NPC飞船如何采矿，同时避免可采矿小行星被过度采掘殆尽。该限制不适用于任务生成的采矿飞船。`

tip "Universal frugal threshold"
	`具备“节俭”性格的飞船仅在自身生命值（护盾 + 船体 —— 即飞船瘫痪时的船体剩余值）低于或等于此百分比，或当前星系内敌对势力明显强于自身时，才会启用消耗弹药或燃料的武器。`

tip "Universal ramscoop"
	`决定是否在所有飞船上启用“通用冲压式集气器”。该集气器会为每艘飞船提供极少量燃料，无论其是否实际安装了冲压式集气器。此集气器的效能衰减远比普通冲压式集气器剧烈，你必须尽可能靠近恒星才能获得可观的燃料补给。启用此项可避免因耗尽燃料且当前星系内未生成任何可获取燃料的NPC而被迫重载存档。`

tip "Minimum arrival distance"
	`所有星系的最小抵达距离。适用于跃迁引擎与跳跃引擎航行。若某星系设定了大于该值的抵达距离，则以该星系设定值为准。若此游戏规则值为“（未设置）”，则始终采用星系自身定义的值。`

tip "Minimum departure distance"
	`所有星系的最小出发距离。飞船仅当距离星系中心超过该出发距离时，方可跃出该星系。若某星系设定了大于该值的出发距离，则以该星系设定值为准。`

tip "Fleet multiplier"
	`星系内随机舰队生成的全局舰队生成速率倍率。`

tip "Lock gamerules"
	`控制是否允许在飞行员创建后修改游戏规则。若设为真，该飞行员的主菜单将不显示‘游戏规则’按钮。若对已有飞行员启用此选项，离开该菜单后将无法再次返回。`

tip "Fighters hit when disabled"
	`控制哪些搭载的飞船（战斗机与无人机）在被禁用状态下，除非被直接瞄准，否则不会被炮弹击中。战斗机和无人机是脆弱的飞船，玩家替换它们可能十分困难或繁琐；此游戏规则旨在提升其生存能力，而不会增强其战斗效能。但被禁用的搭载飞船仍可能被爆炸波及——无论爆炸来自武器还是其他飞船的爆炸。`

tip "Universal ammo stocking"
	`控制你已安装或存于货舱中的大多数副武器弹药，是否可在任意装备店购买（即使该装备店通常并不出售此类弹药）。当此规则启用时，每种副武器的描述中会注明其弹药是否支持在任意地点补给。`
